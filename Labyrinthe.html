Comment jouer ?

But : Ramasse toutes les piles d'énergie 🔋 dispersées sur la carte, puis rejoins le portail de sortie violet 🌀 en utilisant le moins de blocs possible.

1. Ajoute des blocs : Clique sur les instructions du panneau du milieu. Elles s'insèrent dans la zone active.

2. Répète des actions (Boucle 🔄) : Aux niveaux avec le bloc orange Répéter, clique sur le fond de la boucle (il devient orange clair) puis clique sur les actions pour les mettre dedans !

3. Oriente-toi bien : Le robot tourne par rapport à sa propre direction (sa tête 🤖), pas la tienne !

4. Attention aux Dangers : Évite les flaques d'eau 🌊 et les trous de boue 🕳️. Il n'y a pas de murs physiques mais le robot tombera s'il marche dedans ! Si tu fais un détour sans danger, c'est possible, mais ton programme risque de dépasser la limite de blocs.

5. Exécute : Appuie sur le bouton vert Démarrer pour voir le robot s'animer pas-à-pas.
